# EAP Practical Problem Solving

Welcome to this **EAP Practical Problem Solving** repository! This repository is designed for **intermediate-level developers** who want to improve their problem-solving skills using **arrays, strings, recursion, hashing, and math-based problems**. The problems cover a wide range of topics commonly seen in coding interviews, assessments, and competitive programming.  

## Purpose

The main goal of this repository is to provide a **structured and organized set of practice problems** to:  
- Strengthen your **algorithmic thinking**.  
- Improve **coding efficiency** and problem-solving speed.  
- Prepare for **technical interviews** in companies like FAANG and other tech firms.  
- Build confidence in handling **array, string, recursion, and hashing-based problems**.  

## Repository Structure

This repository contains the following Markdown files with categorized problem-solving questions:

1. **[Practical Problem Solving](./PRACTICAL_PROBLEM_SOLVING.md)**  
   - Guideline for developers how to understand and solve problems.  
   - Enhance problem pattern recognition and algorithm thinking. 

2. **[Basic Problem Solving](./BASIC_PROBLEM_SOLVING.md)**  
   - Simple to intermediate problems for arrays, strings, and numbers.  
   - Great for building foundational problem-solving skills.  

3. **[FAANG Problem Solving](./FAANG_PROBLEM_SOLVING.md)**  
   - Higher difficulty problems commonly seen in **FAANG interviews**.  
   - Includes advanced data structures, algorithms, and optimized solutions.  

4. **[Problem Solving Practice](./PROBLEM_SOLVING_PRACTICE.md)**  
   - A mix of problems for regular **practice and revision**.  
   - Covers hashing, frequency maps, recursion, dynamic programming, and array manipulations.
